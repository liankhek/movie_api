Movie API Documentation


Introduction

Welcome to the Movie API documentation. This API is part of the full-stack JavaScript development project, showcasing mastery in technologies like MongoDB, Express, React, and Node.js (MERN stack). The Movie API provides a comprehensive server-side solution for managing movie information, including details about movies, directors, and genres.

Purpose

The primary goal of the Movie API is to serve movie enthusiasts by offering access to information about different movies, directors, and genres. Users can retrieve data on movies, update their profiles, add movies to their favorites, and more.
