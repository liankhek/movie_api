Movie API Documentation


Introduction

Welcome to the Movie API documentation. This API is part of the full-stack JavaScript development project, showcasing mastery in technologies like MongoDB, Express, React, and Node.js (MERN stack). The Movie API provides a comprehensive server-side solution for managing movie information, including details about movies, directors, and genres.

Purpose

The primary goal of the Movie API is to serve movie enthusiasts by offering access to information about different movies, directors, and genres. Users can retrieve data on movies, update their profiles, add movies to their favorites, and more.

Get List of Movies

  • Endpoint URL: /movies
  • Method Type: GET
  • Query Parameters: None
  • Request Body: None
  • Response Data Format: JSON

Get Movie Details by Title

  • Endpoint URL: /movies/:title
  • Method Type: GET
  • Query Parameters: None
  • Request Body: None
  • Response Data Format: JSON

Get Genre Details by Genre Name

  • Endpoint URL: /movies/genre/:genreName
  • Method Type: GET
  • Query Parameters: None
  • Request Body: None
  • Response Data Format: JSON

Get Director Details by Director Name

  • Endpoint URL: /movies/directors/:directorName
  • Method Type: GET
  • Query Parameters: None
  • Request Body: None
  • Response Data Format: JSON

Create New User

  • Endpoint URL: /users
  • Method Type: POST
  • Query Parameters: None
  • Request Body: JSON (Example: {"name": "John Doe"})
  • Response Data Format: JSON (Example: {"id": 1, "name": "John Doe", "favoriteMovies": []})

Update User Information(name)

  • Endpoint URL: /users/:id
  • Method Type: PUT
  • Query Parameters: None
  • Request Body: JSON (Example: {"name": "New Name"})
  • Response Data Format: JSON (Example: {"id": 1, "name": "New Name", "favoriteMovies": []})

Add Movie to User's Favorite List

  • Endpoint URL: /users/:id/:movieTitle
  • Method Type: POST
  • Query Parameters: None
  • Request Body: None
  • Response Data Format: JSON

Remove Movie from User's Favorite List

  • Endpoint URL: /users/:id/:movieTitle
  • Method Type: DELETE
  • Query Parameters: None
  • Request Body: None
  • Response Data Format: JSON

Delete User

  • Endpoint URL: /users/:id
  • Method Type: DELETE
  • Query Parameters: None
  • Request Body: None
  • Response Data Format: JSON
